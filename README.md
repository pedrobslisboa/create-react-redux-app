## INSTALL

- clone this repo:
        git clone https://github.com/pedrobslisboa/create-react-redux-app.git
- istall dependencies:
        npm install
- change the name of the app at package.json:

```diff
   {
-  "name": "create-react-redux-app",
+  "name": "YOUR_APP_NAME_HERE",
   "version": "0.1.0",
   "private": true,

```
